About Me

I am a student at Metland School majoring in PPLG (INFORMATION TECHNOLOGY)with a strong passion for programming and technology. My main focus is Fullstack Web Development, where I build dynamic and responsive websites using React with Vite for the Front-End, and PHP with Laravel, Firebase, and SQL Server on the Back-End. I’m experienced in developing RESTful APIs, implementing authentication systems, and managing databases to support scalable web applications. In mobile development, I create cross-platform apps using React Native integrated with Firebase for real-time features like authentication, database, and push notifications. I also have experience in building native Android apps using Kotlin, especially when deeper platform integration is needed. I'm also passionate about IoT projects, working with Arduino, Python, and C to develop simple automation systems and embedded solutions. Alongside my development experience, I have solid skills in IT Support, including troubleshooting hardware/software issues, system maintenance, device optimization, and user assistance—giving me a strong foundation in both software and hardware areas. I am a self-taught and highly motivated learner, consistently exploring new technologies through online resources and personal projects. My goal is to grow into a versatile and dependable developer with strong technical and problem-solving skills—ready to take on real-world challenges in the tech industry.
